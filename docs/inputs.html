Documentation¶

The following is my best attempt to explain the structure of the package and how it works. The package only includes one module called inputs and it contains all the classes and functions necessary to be able to calculate an exposure time.

Object that represents the target which you wish to observe.

Parameters

  • magnitude (float) – The magnitude of the object which you are observing.

  • mag_system (str) – The magnitude system used to measure the above magnitude.

  • filter_range (tuple) – The range (in Angstroms) which you wish to observe in.

  • sed (Interpolated object, optional) – The spectral energy distribution of the target star. Defaults to None.

  • temp (float, optional) – The temperature of the target star (in Kelvin). Defaults to 5778.
